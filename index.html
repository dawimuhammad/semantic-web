Menjadi Full Stack Web Developer dalam 12 minggu

Kelas Pemrograman Full Stack JavaScript Web Developer di Jakarta, Indonesia

  • HACKTIV8 Coding Bootcamp

  • HACKTIV8 adalah program pembelajaran Full Stack JavaScript immersive pertama di Indonesia. Kurikulum kami dirancang untuk membangun keahlian yang dapat diaplikasikan pada perkembangan dunia teknologi terkini.

  • Lihat Silabus Hacktiv7

  • Full Stack JavaScript Immersive

  • Kelas pengembangan web menggunakan JavaScript yang intensif selama 12 minggu di Jakarta. Kamu akan mampu membuat aplikasi web ataupun mobile, termasuk bagaimana menghubungkannya dengan database, dan API, TDD, hingga melakukan deployment dan scaling terhadap aplikasi kamu.

  • Pelajari Lebih Lanjut

  • Kelas pembelajaran Basic Front End Wed Development menggunakan HTML, CSS, dan Java Script secara part-time di Jakarta. Program ini akan mengajak peserta untuk membuat berbagai proyek yang dapat menjadi portofolio pribadi sebagai Front-End Web Developer.

  • Kamu akan belajar mengenai Agile Concepts, Kanban High Level, SCRUM Details, SCRUM development thinking, knowledge areas, dan teamwork. Program ini akan memberikan kamu pemahaman mendalam akan bagaimana mempraktikkan framework paling populer sekelas Kanban dan SCRUM.
